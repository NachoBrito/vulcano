# Product Context: RAG Frontend

## Why this project exists
The RAG Frontend allows users to easily query the VulcanoDB-powered RAG system through a web interface, demonstrating the database's capabilities in a real-world scenario.

## Problems it solves
- Provides an accessible way to interact with the RAG backend.
- Simplifies the visualization of retrieved documents and generated responses.

## User Experience Goals
- Fast and responsive interface.
- Minimalist design.
- Clear feedback for queries and results.
