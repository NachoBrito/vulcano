# Tech Context: RAG Frontend

## Core Technologies
- **JavaScript (ES6+)**
- **HTML5 / CSS3**

## Development Tools
- **npm**: Dependency management.
- **Vite**: Bundler and dev server (handles resource compaction).
- **ESLint**: Linting and code quality.
- **Prettier**: Code formatting.

## Technical Constraints
- No external frameworks allowed (must be vanilla JS).
- Must be compatible with modern browsers.
